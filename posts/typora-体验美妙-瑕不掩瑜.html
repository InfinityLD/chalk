什么是 Typora ？

Typora 是一款支持实时预览的 Markdown 文本编辑器。它有 OS X, Windows, Linux 三个平台的版本支持，是完全免费的

什么是 Markdown？

Markdown 是一种用于编写结构化文档的语言，使用 Markdown 语言编写文档可以通过指定代码实现文字格式的转换和排版，是一门非常容易入手且效率较高的语言，可以在这篇文章中或者在 Typora 中点击 “帮助” -> “Markdown Reference” 快速入门 Markdown

通过 Typora+Markdown 可以做哪些事情？

我们通常提起写文档，小论文之类的，想到的都是通过 Word 来完成任务。对于文档来说，我们想在尽可能不打断自己思路的情况下，无需太过顾及格式，完成主体内容的创作，而 Word 过多的排版选项，以及对新手不友好的学习成本，会让使用者在输入内容的同时，需要不断地调整格式，使得工作效率较低。

Markdown 可以让使用者只需专心于文本内容的创作，格式问题只需一行或几行代码即可搞定

Markdown 语法

标题 (Headers)

Markdown 中通过使用1-6个 # 来表示一级到六级标题

列表 (Lists)

在 Markdown 中列表分为有序列表和无序列表，无序列表可以使用 * + 或者 - 创建，而有序列表直接使用 1. 2. 创建

将大象塞入冰箱的三个步骤

  1. 打开冰箱门
  2. 把大象塞进去
  3. 关上冰箱门

对了，还可以创建任务列表 (Task List)，区分任务是否完成

  • 好好学习
  • 天天向上
  • 吃喝玩乐

插入图片

Typora 相比于其他 Markdown 编辑器更人性化的地方在于，将插入图片的操作做到了直觉化。根据 Markdown 语法，插入图片可以通过

的格式输入，而在 Typora 中可以直接通过复制粘贴的形式插入

插入数学公式

Typora 中个人最喜欢的一点就是支持 LaTeX 输入数学公式。有了这个功能，我就可以在只使用 Typora 的基础上，加上图片插入等功能，完成论文笔记，项目文档的编写了
